How It Works

Design first, code second

Instead of prompting AI to generate code and hoping it looks right, start with the design. Claude creates, iterates, and perfects visuals in a live canvas. Once it's right, export a spec and let any AI coding tool rebuild it as a real app.

Features

Everything an AI designer needs

38 MCP tools. Zero arbitrary code execution. Full design control.

38 MCP Tools

Elements, styles, pages, viewports, projects, assets, history, and export. Full design control through a controlled API.

Real-Time Viewer

Every change appears instantly via WebSocket deltas. Watch the design build itself in your browser.

Responsive Design

Switch between mobile, tablet, and desktop viewports. Design for any screen size.

Multi-Project

Create and switch between independent design projects. Each with its own git history.

Safe by Design

No arbitrary code execution. The AI can only manipulate designs through controlled MCP tools.

Export Anything

Standalone HTML, structured design specs for AI coding tools, or PNG screenshots from any viewport.

Quick Start

Up and running in 60 seconds

Clone, install, and start designing. Works with Claude Code, Cursor, and VS Code.

Terminal 
# Clone and install
git clone https://github.com/ddalcu/ghost-canvas.git
cd ghost-canvas
npm install

# Start with Claude Code (recommended)
claude

# Load the design skill
/design

# Tell Claude what to design
"Create a landing page for a coffee shop"
